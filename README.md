# 💊 Emergency Drugs — Clinical Reference

> A sleek, dark-themed emergency drug dose calculator and clinical protocol reference built for junior doctors. No backend, no install — just open and use.

---

## 🖥️ Live Demo

> Deploy via GitHub Pages: **Settings → Pages → Deploy from `main` branch → `/root`**  
> Then visit: `https://monish4030.github.io/EMERGENCY-DRUGS`

---

## ✨ Features

- ⚡ **Weight-Based Dose Calculator** — Enter patient weight (kg) and instantly get calculated doses for every drug
- 💊 **Drug Reference Cards** — Expandable cards with mechanism of action, dosing, route, onset, duration, concentration, and preparation instructions
- 📋 **Step-by-Step Treatment Protocols** — Detailed clinical protocols for RSI, cardiac arrest, seizure, anaphylaxis, and more
- 🔍 **Search & Filter** — Search by drug name or filter by category (Sedation, Analgesia, Cardiac, etc.)
- ⚠️ **Warnings & Contraindications** — Each drug panel includes relevant safety warnings
- 📊 **Monitoring Parameters** — Key parameters to monitor after administration
- 🚨 **Emergency Protocols Section** — Structured protocols with linked drug buttons
- 📢 **SBAR Handover Framework** — Built-in SBAR reference for clinical handovers
- 📖 **Quick Reference Guide** — Dosing principles and clinical tips
- 🌑 **Dark UI** — Cyberpunk-style design optimized for low-light clinical environments

## 🚀 Getting Started

This is a **single-file, zero-dependency** web app. No npm, no build step.

### Run Locally

```bash
# Clone the repo
git clone https://github.com/monish4030/EMERGNGY-DRUGS.git

# Open in browser
open index.html
```

Or just double-click `index.html` — it runs entirely in the browser.

### Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch → `main` → `/ (root)`**
4. Click **Save** — your site will be live in ~1 minute
